## 🗂️ Projetos

### 🎬 Screenmatch
Aplicação Java para busca e listagem de filmes e séries, consumindo API REST externa.

**Conceitos aplicados:**
- Orientação a Objetos (POO)
- Herança e Polimorfismo
- Interfaces e `@Override`
- Consumo de API REST com JSON
